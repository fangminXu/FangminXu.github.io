JavaScript简介

JavaScript 是一种轻量级的编程语言。JavaScript 是可插入 HTML 页面的编程代码。JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。

主要作用：

  • 直接写入HTML输出流
  • 对事件进行反应
  • 改变HTML内容、图像、样式
  • 验证输入
